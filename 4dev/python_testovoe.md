# Тестовое задание: Backend-разработка (Python) — Отладка FastAPI приложения

## Описание задания
Необходимо провести отладку приложения для парсинга публичных вакансий с сайта **Selectel**.

**Описание проекта:**
Проект реализован с помощью фреймворка **FastAPI** и выполняет следующие функции:
- Осуществляет парсинг вакансий каждые 5 минут.
- Сохраняет информацию в базе данных **PostgreSQL**.
- Предоставляет **CRUD API** для операций с вакансиями (чтение, добавление, изменение, удаление).

Подробную информацию о возможностях приложения, его настройках и алгоритме запуска можно найти в файле `README` проекта.

**Ссылка на исходный код:**
[archive_scw.zip](http://files.selectel.ru/docs/ru/archive_scw.zip)

**Задача:**
В приложении намеренно допущены баги, препятствующие корректной работе. Необходимо запустить приложение локально, выявить и исправить умышленно внедренные **8 багов** разной степени сложности.

Для засчитывания задания необходимо обеспечить выполнение следующих условий:

1.  Успешный запуск приложения без ошибок.

4.  Корректная работа фоновой задачи парсинга по заявленному расписанию (каждые 5 минут).
